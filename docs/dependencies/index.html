Python

As of the moment, ATOM supports Python 3.6, 3.7 and 3.8.

Packages

ATOM is built on top of several existing Python libraries. The required packages are necessary for it's correct functioning. Additionnaly, you can install some optional packages needed for specific methods or to use machine learning models not provided by sklearn.

Required

  • numpy (>=1.17.2)
  • scipy (>=1.4.1)
  • pandas (>=1.0.3)
  • scikit-learn (>=0.23.1)
  • category-encoders (>=2.1.0)
  • tabulate (>=0.8.6)
  • tqdm (>=4.35.0)
  • typeguard (>=2.7.1)
  • scikit-optimize (>=0.7.4)
  • matplotlib (>=3.1.0)
  • seaborn (>=0.9.0)

Optional

  • pandas-profiling (>=2.3.0)
  • imbalanced-learn (>=0.5.0)
  • gplearn (>=0.4.1)
  • xgboost (>=0.90)
  • lightgbm (>=2.3.0)
  • catboost (>=0.19.1)
